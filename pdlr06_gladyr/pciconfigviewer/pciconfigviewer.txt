Драйвер				-	pciconfigviewer.sys 
Символічне посилання	-	\\.\pciconfigviewer  
Інтерфейс			-	DeviceIoControl
Підтримувані IOCTL-запити: 
IOCTL_GET_PCI_COMMON_HEADER (0x22E000) - читання регістрів
конфігураційного простору абонента шини PCI у буферизованому
режимі	
Вхід:
lpInBuffer - адреса буфера з подвійним словом, що визначає
		координати пристрою на шині (див. структуру PCI_SLOT_NUMBER_EXT): 
31 30         24 23            16 15                11 10                     8 7                        2 1 0
 = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = =
 |1|Reserved|Bus number|Device number|Function number|Register number|0|0|
 = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = = =
lpOutBuffer - адреса буферу розміром не менше ніж 64 байти 
Вихід: У буфері за адресою lpOutBuffer повертаються значення
16-ти 4-байтових регістрів від 0 до 3Сh конфігураційного простору
пристрою із зазначеними координатами. Для неіснуючих пристроїв
повертається 0xFFFFFFFF.
